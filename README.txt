PAINEL LOCAL DO BOT - VERSAO COM FORMULARIO

O QUE TEM
- bot do WhatsApp com Baileys
- IA via Groq
- painel local com formulários
- botão para adicionar/remover produto
- botão para adicionar/remover serviço
- edição simples da configuração da IA

COMO USAR
1) instale Node.js e Git
2) abra app/.env
3) coloque sua chave em GROQ_API_KEY
4) dê duplo clique em iniciar-app.bat
5) abra http://localhost:3000

NO PAINEL
- cadastre seus produtos
- cadastre seus serviços
- edite o tom de atendimento
- edite a mensagem de boas-vindas
- edite regras e informações fixas
- teste a resposta antes de usar no WhatsApp

SE QUISER PAREAR DE NOVO
apague a pasta:
app/auth_info_baileys
